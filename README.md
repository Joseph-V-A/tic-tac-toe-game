# 🧠 Tic-Tac-Toe Game Project
The Tic-Tac-Toe Game Project is a simple console-based game implemented in C, allowing two players to play against each other. The game features a text-based interface, where players enter numbers between 1 and 9 to make moves on the board. The game checks for winning conditions after each move and announces the winner if the game is won. If the game is not won after 9 moves, it is declared a draw, and the game starts over.

## 🚀 Key Features
* Simple console-based interface
* Two-player gameplay
* Valid move checking to prevent overwriting existing moves
* Winning condition checking after each move
* Draw condition checking after 9 moves
* Option to play again after each game

## 🛠️ Tech Stack
* Programming Language: C
* Libraries:
	+ `stdio.h` for input/output operations
	+ `conio.h` for the `getch` function
* Compiler: GCC (or any other C compiler)
* Platform: Console-based, can run on any operating system with a C compiler

## 📦 Installation
To install and run the Tic-Tac-Toe Game Project, follow these steps:
### Prerequisites
* C compiler (e.g., GCC)
* Console or terminal emulator
### Installation
1. Clone the repository or download the source code
2. Navigate to the project directory in the console or terminal
3. Compile the `index.c` file using the C compiler (e.g., `gcc index.c -o tic_tac_toe`)
4. Run the compiled executable (e.g., `./tic_tac_toe` on Linux/macOS or `tic_tac_toe.exe` on Windows)

## 💻 Usage
To play the game, simply run the executable and follow the prompts. Enter numbers between 1 and 9 to make moves on the board. The game will check for winning conditions after each move and announce the winner if the game is won. If the game is not won after 9 moves, it is declared a draw, and the game starts over.
